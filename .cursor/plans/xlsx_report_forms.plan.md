# XLSX: формы отчётов — отложено

**Статус:** не в текущем скоупе. См. [docx_parser_corpus_plan_af28c357.plan.md](./docx_parser_corpus_plan_af28c357.plan.md).

## Зачем отдельно

В архиве ~1345 `.xlsx` (формы и заполненные отчёты). Они нужны для:
- выгрузки шаблона отчёта ДЗО из распарсенных мер;
- импорта заполненного отчёта в `Response`;
- (косвенно) обучающих меток «мера → подразделение» — датасет для фазы 51 DOCX-плана, **без** реализации Excel в продукте.

## Будущие фазы (черновик)

| Фаза | Содержание |
|------|------------|
| 60 | `xlsx` / `exceljs`, `lib/report-forms/generate-xlsx.ts` — колонки как в форме 6837 |
| 61 | Фильтр строк по подразделению / тегам меры (сеть, SIEM, почта) |
| 62 | Импорт заполненного xlsx → prefill `Response` по строкам |
| 63 | (опционально) автоотправка на `otd93@fstec.ru` |

## Зависимости

- Стабильные коды и тексты мер из DOCX-парсера
- Рекомендации маршрутизации (фаза 51 DOCX-плана) — чтобы строки формы совпадали с поручениями
